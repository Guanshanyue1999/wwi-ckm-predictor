# WWI CKM Predictor

Streamlit app for predicting CKM risk using multiple central obesity indices and an XGBoost model.

## Demo
https://wwi-ckm-predictor.streamlit.app/

## Requirements
- Python 3.10+

## Setup
1. Install dependencies:
   pip install -r requirements.txt
2. Run the app:
   streamlit run app.py

## Files
- app.py: Streamlit UI and inference
- model_meta.json / best_model_xgboost.pkl / scaler.pkl: model artifacts
- requirements.txt: runtime dependencies

## Notes
This repository contains only the application code and assets required to run the demo.
